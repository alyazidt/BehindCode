📄 appsettings.json

الحين في ملف appsettings.json نضيف هذا الجزء:


{
  "Logging": {
    "LogLevel": {
      "Default": "Information",
      "Microsoft.AspNetCore": "Warning"
    }
  },
  "AllowedHosts": "*",
  "ConnectionStrings": {
    "Connection": "Data Source=.;Database=Eventom;Integrated Security=True;Connect Timeout=30;Encrypt=True;Trust Server Certificate=True;Application Intent=ReadWrite;Multi Subnet Failover=False"
  }
}

📝 شرح كل جزء:

  • Data Source=. تعني أن قاعدة البيانات على نفس الجهاز.
  • Database=Eventom اسم قاعدة البيانات.
  • Integrated Security=True يستخدم بيانات دخول الويندوز بدون اسم مستخدم وكلمة سر.
  • Connect Timeout=30 يمهل التطبيق ٣٠ ثانية للاتصال قبل أن يفشل.
  • Encrypt=True يفعّل التشفير لحماية الاتصال.
  • TrustServerCertificate=True يقبل الشهادات حتى لو غير موقعة رسميًا.
  • ApplicationIntent=ReadWrite يسمح بالقراءة والكتابة على القاعدة.
  • MultiSubnetFailover=False لا يستخدم التبديل بين الشبكات (يُستخدم في بيئات متقدمة).
